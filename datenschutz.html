Datenschutzerklärung

1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können.

2. Hosting

Diese Website wird bei einem externen Dienstleister gehostet (Hoster). Die personenbezogenen Daten, die auf dieser Website erfasst werden, werden auf den Servern des Hosters gespeichert. Hierbei kann es sich v.a. um IP-Adressen, Kontaktanfragen, Meta- und Kommunikationsdaten, Vertragsdaten, Kontaktdaten, Namen, Websitezugriffe und sonstige Daten, die über eine Website generiert werden, handeln.

3. AIMOS Software — Datenverarbeitung

AIMOS ist eine lokal betriebene Software. Die AIMOS-Software selbst sendet keine Daten an uns oder Dritte. Alle Daten verbleiben auf dem Server des Betreibers (SovereignNode).

Datenverarbeitung durch AIMOS

  • ✓ LLM-Inferenz: Vollständig lokal auf dem SovereignNode
  • ✓ Agenten-Gedächtnis: Lokale SQLite-Datenbank pro Agent
  • ✓ Chat-Verläufe: Lokale PostgreSQL-Datenbank
  • ✓ Audit-Logs: Lokales Dateisystem
  • ⚠ Telegram-Nachrichten: Transit über Telegram-Server (standardmäßiger Telegram-Transport)
  • ⚠ Externe LLMs (optional): Nur bei expliziter Eskalation, automatisch anonymisiert durch PII-Vault

4. PII-Vault (Anonymisierung)

Wenn AIMOS eine externe KI-API kontaktiert (z.B. für komplexe Analyse-Aufgaben), werden alle personenbezogenen Daten vor der Übertragung automatisch anonymisiert. Namen, E-Mail-Adressen, Telefonnummern und Zugangsdaten werden durch Platzhalter ersetzt. Die Zuordnung (Platzhalter → Originalwert) verbleibt ausschließlich auf dem lokalen Server.

5. Ihre Rechte

Sie haben jederzeit das Recht auf unentgeltliche Auskunft über Ihre gespeicherten personenbezogenen Daten, deren Herkunft und Empfänger und den Zweck der Datenverarbeitung sowie ein Recht auf Berichtigung oder Löschung dieser Daten. Hierzu sowie zu weiteren Fragen zum Thema personenbezogene Daten können Sie sich jederzeit an uns wenden.

6. Analyse-Tools und Werbung

Diese Website verwendet keine Analyse-Tools, Tracking-Cookies oder Werbenetzwerke. Es werden keine Daten an Google Analytics, Facebook Pixel oder vergleichbare Dienste übertragen.

7. SSL-Verschlüsselung

Diese Seite nutzt aus Sicherheitsgründen eine SSL-Verschlüsselung. Eine verschlüsselte Verbindung erkennen Sie daran, dass die Adresszeile des Browsers von "http://" auf "https://" wechselt und an dem Schloss-Symbol in Ihrer Browserzeile.

8. Verantwortliche Stelle

Die verantwortliche Stelle für die Datenverarbeitung auf dieser Website ist im Impressum angegeben.
