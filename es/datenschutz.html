Política de privacidad

1. Protección de datos de un vistazo

Indicaciones generales

Las siguientes indicaciones ofrecen una visión general de lo que sucede con sus datos personales cuando visita este sitio web. Los datos personales son todos aquellos datos con los que usted puede ser identificado personalmente.

2. Alojamiento

Este sitio web está alojado por un proveedor de servicios externo (hosting). Los datos personales recogidos en este sitio web se almacenan en los servidores del proveedor de hosting. Estos pueden incluir, entre otros, direcciones IP, solicitudes de contacto, metadatos y datos de comunicación, datos contractuales, datos de contacto, nombres, accesos al sitio web y otros datos generados a través del sitio web.

3. Software AIMOS — Tratamiento de datos

AIMOS es un software de operación local. El software AIMOS en sí no envía ningún dato a nosotros ni a terceros. Todos los datos permanecen en el servidor del operador (SovereignNode).

Tratamiento de datos por AIMOS

  • ✓ Inferencia LLM: completamente local en el SovereignNode
  • ✓ Memoria del agente: base de datos SQLite local por agente
  • ✓ Historial de conversaciones: base de datos PostgreSQL local
  • ✓ Registros de auditoría: sistema de archivos local
  • ⚠ Mensajes de Telegram: tránsito a través de servidores de Telegram (transporte estándar de Telegram)
  • ⚠ LLMs externos (opcional): solo en caso de escalado explícito, anonimizados automáticamente mediante PII-Vault

4. PII-Vault (Anonimización)

Cuando AIMOS contacta una API de IA externa (por ejemplo, para tareas de análisis complejas), todos los datos personales se anonimizan automáticamente antes de la transmisión. Nombres, direcciones de correo electrónico, números de teléfono y credenciales de acceso se reemplazan por marcadores de posición. La asignación (marcador de posición → valor original) permanece exclusivamente en el servidor local.

Detalles: Documentación de Compliance

5. Sus derechos

Usted tiene en todo momento el derecho a obtener información gratuita sobre sus datos personales almacenados, su origen y destinatarios, así como la finalidad del tratamiento de datos, y el derecho a la rectificación o eliminación de dichos datos. Para este fin, así como para cualquier otra consulta relativa a datos personales, puede dirigirse a nosotros en cualquier momento.

6. Herramientas de análisis y publicidad

Este sitio web no utiliza herramientas de análisis, cookies de seguimiento ni redes publicitarias. No se transmiten datos a Google Analytics, Facebook Pixel ni servicios similares.

7. Cifrado SSL

Este sitio utiliza cifrado SSL por motivos de seguridad. Puede reconocer una conexión cifrada cuando la barra de direcciones del navegador cambia de "http://" a "https://" y aparece el símbolo del candado en la barra del navegador.

8. Entidad responsable

La entidad responsable del tratamiento de datos en este sitio web se indica en el Aviso legal.
