Politique de confidentialité

1. Protection des données en bref

Informations générales

Les informations suivantes vous donnent un aperçu de ce qui arrive à vos données personnelles lorsque vous visitez ce site web. Les données personnelles sont toutes les données permettant de vous identifier personnellement.

2. Hébergement

Ce site web est hébergé par un prestataire externe (hébergeur). Les données personnelles collectées sur ce site sont stockées sur les serveurs de l'hébergeur. Il peut s'agir notamment d'adresses IP, de demandes de contact, de métadonnées et de données de communication, de données contractuelles, de coordonnées, de noms, d'accès au site et d'autres données générées par un site web.

3. Logiciel AIMOS — Traitement des données

AIMOS est un logiciel exploité localement. Le logiciel AIMOS lui-même n'envoie aucune donnée vers nous ou vers des tiers. Toutes les données restent sur le serveur de l'exploitant (SovereignNode).

Traitement des données par AIMOS

  • ✓ Inférence LLM : entièrement locale sur le SovereignNode
  • ✓ Mémoire des agents : base de données SQLite locale par agent
  • ✓ Historiques de conversation : base de données PostgreSQL locale
  • ✓ Journaux d'audit : système de fichiers local
  • ⚠ Messages Telegram : transit via les serveurs Telegram (transport Telegram standard)
  • ⚠ LLM externes (optionnel) : uniquement en cas d'escalade explicite, anonymisation automatique par PII-Vault

4. PII-Vault (Anonymisation)

Lorsqu'AIMOS contacte une API d'IA externe (p. ex. pour des tâches d'analyse complexes), toutes les données personnelles sont automatiquement anonymisées avant la transmission. Les noms, adresses e-mail, numéros de téléphone et identifiants sont remplacés par des marqueurs. La correspondance (marqueur → valeur originale) reste exclusivement sur le serveur local.

Détails : Documentation de conformité

5. Vos droits

Vous avez à tout moment le droit d'obtenir gratuitement des informations sur vos données personnelles stockées, leur origine, leurs destinataires et la finalité du traitement, ainsi qu'un droit de rectification ou de suppression de ces données. Pour cela et pour toute question relative aux données personnelles, vous pouvez nous contacter à tout moment.

6. Outils d'analyse et publicité

Ce site web n'utilise aucun outil d'analyse, cookie de suivi ou réseau publicitaire. Aucune donnée n'est transmise à Google Analytics, Facebook Pixel ou des services similaires.

7. Chiffrement SSL

Ce site utilise un chiffrement SSL pour des raisons de sécurité. Vous reconnaissez une connexion chiffrée au fait que la barre d'adresse du navigateur passe de « http:// » à « https:// » et au symbole de cadenas dans la barre de votre navigateur.

8. Organisme responsable

L'organisme responsable du traitement des données sur ce site est indiqué dans les Mentions légales.
