Informativa sulla privacy

1. Privacy in sintesi

Informazioni generali

Le seguenti indicazioni forniscono una panoramica su cosa accade ai Suoi dati personali quando visita questo sito web. I dati personali sono tutti i dati con cui Lei può essere identificato/a personalmente.

2. Hosting

Questo sito web è ospitato presso un fornitore di servizi esterno (hoster). I dati personali raccolti su questo sito vengono memorizzati sui server dell'hoster. Questi possono includere indirizzi IP, richieste di contatto, metadati e dati di comunicazione, dati contrattuali, dati di contatto, nomi, accessi al sito e altri dati generati attraverso un sito web.

3. Software AIMOS — Trattamento dei dati

AIMOS è un software gestito localmente. Il software AIMOS stesso non invia alcun dato a noi o a terzi. Tutti i dati rimangono sul server dell'operatore (SovereignNode).

Trattamento dei dati da parte di AIMOS

  • ✓ Inferenza LLM: completamente locale sul SovereignNode
  • ✓ Memoria degli agenti: database SQLite locale per ogni agente
  • ✓ Cronologia chat: database PostgreSQL locale
  • ✓ Log di audit: file system locale
  • ⚠ Messaggi Telegram: transito attraverso i server Telegram (trasporto standard di Telegram)
  • ⚠ LLM esterni (opzionale): solo in caso di escalation esplicita, anonimizzati automaticamente tramite PII-Vault

4. PII-Vault (anonimizzazione)

Quando AIMOS contatta un'API di intelligenza artificiale esterna (ad es. per compiti di analisi complessi), tutti i dati personali vengono anonimizzati automaticamente prima della trasmissione. Nomi, indirizzi e-mail, numeri di telefono e credenziali di accesso vengono sostituiti da segnaposto. La mappatura (segnaposto → valore originale) rimane esclusivamente sul server locale.

Dettagli: Documentazione Compliance

5. I Suoi diritti

Lei ha in qualsiasi momento il diritto di ottenere gratuitamente informazioni sui Suoi dati personali memorizzati, sulla loro origine e sui destinatari, nonché sullo scopo del trattamento dei dati, e il diritto di rettifica o cancellazione di tali dati. Per questo e per ulteriori domande sul tema dei dati personali può contattarci in qualsiasi momento.

6. Strumenti di analisi e pubblicità

Questo sito web non utilizza strumenti di analisi, cookie di tracciamento o reti pubblicitarie. Nessun dato viene trasmesso a Google Analytics, Facebook Pixel o servizi analoghi.

7. Crittografia SSL

Questo sito utilizza per motivi di sicurezza una crittografia SSL. Una connessione crittografata si riconosce dal fatto che la barra degli indirizzi del browser passa da "http://" a "https://" e dall'icona del lucchetto nella barra del browser.

8. Titolare del trattamento

Il titolare del trattamento dei dati su questo sito web è indicato nell'avviso legale.
