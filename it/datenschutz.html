Informativa sulla privacy

1. Privacy in sintesi

Informazioni generali

Le seguenti informazioni forniscono una panoramica di ciò che accade ai Suoi dati personali quando visita questo sito web. I dati personali sono tutti i dati con cui è possibile identificarLa personalmente.

2. Hosting

Questo sito web è ospitato da un fornitore di servizi esterno (hoster). I dati personali raccolti su questo sito vengono memorizzati sui server dell'hoster. Si tratta in particolare di indirizzi IP, richieste di contatto, metadati e dati di comunicazione, dati contrattuali, dati di contatto, nomi, accessi al sito web e altri dati generati attraverso un sito web.

3. Software AIMOS — Trattamento dei dati

AIMOS è un software gestito localmente. Il software AIMOS stesso non invia alcun dato a noi o a terzi. Tutti i dati rimangono sul server dell'operatore (SovereignNode).

Trattamento dei dati tramite AIMOS

  • ✓ Inferenza LLM: completamente locale sul SovereignNode
  • ✓ Memoria degli agenti: database SQLite locale per agente
  • ✓ Cronologia chat: database PostgreSQL locale
  • ✓ Log di audit: file system locale
  • ⚠ Messaggi Telegram: transito attraverso i server Telegram (trasporto standard Telegram)
  • ⚠ LLM esterni (opzionali): solo in caso di escalation esplicita, anonimizzati automaticamente tramite PII-Vault

4. PII-Vault (Anonimizzazione)

Quando AIMOS contatta un'API esterna di IA (ad es. per attività di analisi complesse), tutti i dati personali vengono anonimizzati automaticamente prima della trasmissione. Nomi, indirizzi e-mail, numeri di telefono e credenziali di accesso vengono sostituiti da segnaposto. L'associazione (segnaposto → valore originale) rimane esclusivamente sul server locale.

Dettagli: Documentazione sulla conformità

5. I Suoi diritti

Ha il diritto in qualsiasi momento di ricevere gratuitamente informazioni sui Suoi dati personali memorizzati, sulla loro origine e destinazione e sullo scopo del trattamento dei dati, nonché il diritto alla rettifica o cancellazione di tali dati. Per questo e per ulteriori domande sul tema dei dati personali può contattarci in qualsiasi momento.

6. Strumenti di analisi e pubblicità

Questo sito web non utilizza alcuno strumento di analisi, cookie di tracciamento o reti pubblicitarie. Nessun dato viene trasmesso a Google Analytics, Facebook Pixel o servizi analoghi.

7. Crittografia SSL

Per motivi di sicurezza questo sito utilizza la crittografia SSL. Una connessione crittografata si riconosce dal fatto che la barra degli indirizzi del browser passa da "http://" a "https://" e dal simbolo del lucchetto nella barra del browser.

8. Titolare del trattamento

Il titolare del trattamento dei dati su questo sito web è indicato nell' Avviso legale.
